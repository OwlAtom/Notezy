# Notezy

## Project setup

```
npm install
```

### Firebase emulators

make sure to install firebase-tools by running `npm install -g firebase-tools`

Also run `firebase login` first

and install java from https://www.oracle.com/java/technologies/downloads/
(you need at least java 11)

```
npm run emulators
```
